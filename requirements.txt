# System Requirements
Node.js (v18.0.0 or higher recommended)
npm (v8.0.0 or higher)

# Backend Dependencies (Node.js/Express)
bcryptjs >= 3.0.3
cors >= 2.8.6
dotenv >= 17.4.2
express >= 5.2.1
jsonwebtoken >= 9.0.3
nodemailer >= 9.0.3
sqlite3 >= 6.0.1

# Frontend Dependencies (React/Vite)
react >= 19.2.7
react-dom >= 19.2.7
react-router-dom >= 7.18.1
lucide-react >= 1.23.0

# Build Tools & Dev Dependencies
vite >= 8.1.1
@vitejs/plugin-react >= 6.0.3
oxlint >= 1.71.0
@types/react >= 19.2.17
@types/react-dom >= 19.2.3

# Database
SQLite3 (Built-in via sqlite3 npm package)

# Notes
- To install backend dependencies, navigate to /Backend and run `npm install`
- To install frontend dependencies, navigate to /Frontend and run `npm install`
- No Python environment is required for this project, despite the presence of this requirements.txt file.
